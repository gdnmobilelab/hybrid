ServiceWorkerInstance

@objc public class ServiceWorkerInstance : NSObject, ServiceWorkerInstanceExports

The core of our service worker functionality. This class wraps a JSContext with a series of helper functions that add our SW helper library, as well as provide hooks for things like setTimeout and event execution
