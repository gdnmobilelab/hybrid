StoredPushEvent

@objc class StoredPushEvent : NSObject, NSCoding

We can’t act on these push payloads immediately because the notification extension doesn’t have enough memory to launch a service worker. Instead, we store the payload to be accessed the next time we fire up a worker, either in the notification content controller, or in the app itself.
